These new commands allow for the execution of bulk insert/update/delete operations. There are two types of bulk operations:

  1. 有序的批量操作，顺序执行所有操作，一旦出错就停止。
  2. 无序批量操作，执行所有的操作，无序的批量操作并不能保证执行顺序。
